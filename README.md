# AddOne (AAD1) — Sample Business Plan

A single-page, self-contained web presentation of a **sample, illustrative business
plan** for the brand **AddOne (stylized AAD1)** — a direct-to-consumer active-lifestyle
and accessories concept.

🔗 **Live site:** https://clarkrobertnye-svg.github.io/addone-aad1-plan/

## What this is

- `index.html` — the published presentation. A mobile-first, self-contained HTML page
  (no build step, no external dependencies) styled with a dark "stained-glass" theme.
- `AddOne_AAD1_Sample_Business_Plan.md` — the original Markdown source the page was
  generated from, kept for reference and future edits.

## Important — not advice

This is a **sample / working draft** prepared for discussion. **The financials are
illustrative placeholders, not projections.** Nothing here is **legal or financial
advice**. Entity formation, trademark, capital, and similar decisions should involve a
startup attorney, a trademark attorney, and an accountant.

## Hosting

Served via **GitHub Pages** from the `main` branch root. Editing `index.html` and pushing
to `main` republishes the live site automatically.
